# Dao.AI.Prompting

A multi-target .NET library for serializing objects and structured data into Markdown, designed for AI prompt engineering and LLM workflows. Easily convert C# objects, dictionaries, and collections into readable Markdown for use in AI systems, chatbots, or documentation.

## Compatibility

| Target Framework | Supported |
|-----------------|-----------|
| .NET 8          | ✅        |
| .NET 9          | ✅        |
| .NET 10         | ✅        |

## Features

- **Markdown Serialization**: Convert any object, dictionary, or collection into Markdown with customizable options.
- **Structured Prompt Support**: Compose system, user, and closing prompts with embedded structured data.
- **Simple API**: Minimal setup, easy integration into any supported .NET project.
